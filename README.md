# Video  Player 

## 1 简介 

Video  Player是一款借助于FFmpeg视频库，进行视频和音频播放的软件，支持视频和音频的播放传输以及视频的快进，音量调节等功能，同时也会视频WebRTC等实时语音和视频传输，并完成跨平台的支持。

## 2 功能介绍  

1. 开始/暂停  
2. 音量+/-  
3. 视频列表  
4. 视频快进/快退 
5. 播放加速/减速  
6. 视频格式转化  
7. 实时传输WebRTC    
8. 支持Windows，Linux，Android等平台
